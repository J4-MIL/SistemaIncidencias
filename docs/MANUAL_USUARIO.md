# Manual de Usuario

1. Ingrese a login.html.
2. Use un usuario de prueba o uno creado por el administrador.
3. Según el rol, el sistema redirige al panel correspondiente.
4. El empleado puede crear y consultar incidencias.
5. El administrador puede registrar usuarios, revisar incidencias y asignar técnicos.
6. El técnico puede atender incidencias asignadas y registrar seguimientos.
